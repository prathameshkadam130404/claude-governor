Privacy Policy

Governor plugin for Claude Code · Effective 2026-07-11

Summary

Governor collects no data, sends no data, and has no servers. It is a fully local, zero-dependency, open-source tool. There is no telemetry, no analytics, no account, and no network communication of any kind initiated by the plugin.

What Governor does with data

Governor reads session metadata that Claude Code already provides locally on your machine — context-window usage, rate-limit percentages and reset times, model name, and session cost — and writes derived state to files on your own machine only:

  • ~/.claude/governor/ — budget snapshots, burn-rate history, runtime state, diagnostics log, pre-compaction transcript archives
  • <your project>/.governor/ — a journal of tool-call names/targets, preserved subagent outputs, and resume checkpoints
  • ~/.claude/settings.json — one statusline entry, written only via the explicit /governor:install step, with a timestamped backup created first

These files never leave your machine unless you copy or commit them yourself. The project directory is self-ignored from git by default (a nested .gitignore), so governor artifacts are not committed accidentally.

What Governor does NOT do

  • No telemetry, analytics, crash reporting, or usage tracking
  • No network requests — the plugin has zero runtime dependencies and no HTTP code
  • No collection of prompts, code, credentials, or personal information
  • No accounts, cookies, or identifiers

Third parties

Governor runs inside Claude Code and only processes data Claude Code exposes locally. Your relationship with Anthropic (including what Claude Code itself transmits) is governed by Anthropic's Privacy Policy, not this one. This documentation site is hosted on GitHub Pages, which is governed by GitHub's Privacy Statement; the site itself contains no trackers.

Data removal

Everything is plain files you own. To remove all governor data: uninstall the plugin, delete ~/.claude/governor/ and any project .governor/ directories, and restore the settings backup (or remove the statusLine key). Nothing persists anywhere else.
